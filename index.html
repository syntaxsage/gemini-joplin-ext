How It Works

1

Configure Settings

Add your Gemini API key and Joplin Web Clipper token in the extension settings.

2

Right-Click to Summarize

Highlight text or right-click any page and select "Summarize" from the context menu.

3

Review & Edit

The AI generates a summary. Preview it, make edits if needed, then save.

4

Auto Save to Joplin

Your summary is automatically saved to Joplin with the source URL and full attribution.

What You Need

🔑 Required Services

  • Google Gemini API Key — Free tier available at ai.google.dev
  • Joplin Instance — Running locally with Web Clipper enabled (localhost:41184)
